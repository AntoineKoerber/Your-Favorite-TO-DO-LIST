# Your Favorite To-Do List

A clean, interactive to-do list application with a beautiful gradient UI. Add tasks, mark them as complete, and delete them with a simple, intuitive interface.

## Features

- **Add Tasks**: Quick task entry with Enter key support
- **Complete Tasks**: Click "Done" to mark tasks with strikethrough
- **Delete Tasks**: Remove completed tasks from the list
- **Word Count**: Real-time word count for preset tasks
- **Beautiful UI**: Animated gradient background with modern styling
- **Responsive Design**: Works on desktop and mobile devices

## Demo

The application features:

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom animations and styling
- **JavaScript** - Vanilla JS for interactivity
- **Bootstrap 5** - Responsive layout and components

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AntoineKoerber/Your-Favorite-TO-DO-LIST.git
   ```

2. Open `index.html` in your browser.

3. No installation or build process required.

## Project Structure

```
├── index.html          # Main application
└── static/
    ├── css/
    │   └── styles.css  # Custom styling
    └── js/
        └── main.js     # Application logic
```

## Usage

1. **Add a Task**: Type in the input field and click "Add" or press Enter
2. **Complete a Task**: Click the "Done" button next to any task
3. **Delete a Task**: After marking as done, click again to delete

## Customization

- Modify colors in `static/css/styles.css`
- Adjust gradient animation timing
- Customize button styles and layout
